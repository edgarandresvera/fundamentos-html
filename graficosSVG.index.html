etiqueta svg

para crear un grafico svg,tenemos dos opciones, la primera es referenciar a traves de un vinculo la imagen ".svg" generada por algun software externo
como por ejemplo incascape o ilustratorbr y la segunda es crear nuestro propio espacio de trabajo y graficar utilizando las herramientas que nos da el navegador y el lenguaje html5.
en caso de incluir un archivo .svg construido previamente lo podemos hacer, por meido de una etiqueta "image" la cual se define dentro de la etiqueta svg.
image es diferente a la etiqueta img es propia de svg, y tiene un atributo que reemplaza al tradicional "src" que es xlink:href
el cual recibe como valor la ruta y nombre del archivo. adicionalmente tambien cuenta como atributos para posisionar la imagen en los puntos "X y Y"
para incluir un archivo svg externo no, es necesario hacerlo dentro de la etiqueta svg.brtambien puedes hacerlo con la etiqueta tradicional"IMG"
si queremos definir nuetro propio espaico de trabajo para empezar a crear graficos,debemos indicar un atributo llamado viewBox dentro de la
etiqueta svg, este atributo sirve para crear un sub-espacio de trabajo,
dentro del cual se generan los elementos de la misma etiqueta, esto quiere decir que con svg, definimos el ancho y el alto, de un tablero principal, y dentro de ese tablero tenemos otro que puede ocupar solo una parte
o todo el espacio que corresponde al "viewBox"
"viewBox" recibe 4 valores numericos separados por espacios o comas, el primero y el segundo valor corresponden al punto donde se situa el segundo tablero,en los respectivos ejes "X y X" el tercero y cuarto valor,
corresponden al ancho y alto en este ejemplo el "viewBox" o espacio de trabajo ocupa todo el espacio del ttrablero. ejemplos para cada forma de trabajo svg.
