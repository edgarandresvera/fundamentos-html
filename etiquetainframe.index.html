en esta ocacion vamos a aprender a inscruptar paginas externas.dentro de una pagina propia, en el marco se vera la pagina que queramos metere.
para lograrlo, insertamos la etiqueta iFrame src ="https://www.ejemplo.com con los valores ancho y alto utilizamos width="400px" height="400px
adicionalmente antes de cerrar la etiqueta iFrame colocamos el atributo "name" que sirve para definir un nombre que sirve por si la pagina no puede ser cargada.
hay que verificar que la pagina no tenga un POP-UP que son paginas que se abren en el navegador y ofrecen interacciones con el cliente para evitar esto, agregamos el atributo SANDBOX


el atributo sandbox tiene los siguientes valores posibles:
  • sandbox="allow-forms" premite que los formularios al interior de el iFrame envien datos si el cliente llena la informacion
    que le soliciten, en caso de no indicar este atributos el cliente puede llenar los datos del formulario, pero cuando
    queira enviar los datos, no enviara nada

  • sandbox="PopUps" permite que se ejecuten ventanas emergentes, si el sitio que ejecutamos en el iFrame las contenga, de esta manera
    si el sitio que vemos en el iFrame tiene ventanas emergentes el navegador las ejecuta sin inconvenientes, si no indicamos este atributo,
    y la pagina alojada en el iFrame tiene ventnas Pop-ups , el navegador las bloqueara y las ejecutara

  • sandbox="alow-scripts" este valor peromite que el navegador ejecute codigos Javascript del lado del clinete, y la posibilidad de acceder a la
    webcam , nuestros archivos y demas dispositivos que tengamos conectados a la computadora. en caso de no indicar el atriburo, el sitio alojado no
    ejecuta los codigos de este tipo

  • sandbox="allow-top-navigation" se usa para permitir que el sitio alojado en el iFrame, atraves de uno de sus enlaces pueda remplazar nuestro
    contenido por otro propuesto por el mismo
    es decir contenido dieferente al nuestro. si omitimos esta linea evitamos este inconveniente.

  • finalmente el atributo "src" muestra la ruta de lo que deseamos mostrar. junto con este atributo tenenmos el "srcdoc" que permite agregar directamente al iFrame codigo html.
